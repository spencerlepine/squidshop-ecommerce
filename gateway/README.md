# SquidShop Gateway Service

NodeJS, Express, and Docker API. CI/CD using GitHub Actions.
## Setup
```sh
$ docker-compose up -d
# access on localhost:5000
```

## Development
```sh
$ yarn install
$ yarn run start
$ yarn run test
$ yarn run lint
```

## Linter
Uses personal [@spencerlepine](https://github.com/spencerlepine/lint-config) lint configuration.
